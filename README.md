- 34：**二分模板**

  ```java
  //[l,mid][mid+1,l]
  while (l < r) {
      mid = (l + r) >> 1;
      if (check(mid)) {
          r = mid;
      } else {
          l = mid + 1;
      }
  }
  
  //[l,mid-1][mid,r]
  while (l < r) {
      mid = (l + r + 1) >> 1;
      if (check(mid)) {
          l = mid;
      } else {
          r = mid-1;
      }
  }
  
  ```
